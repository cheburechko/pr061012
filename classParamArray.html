Detailed Description

Assoﾑ（ative array (keys are strings, values are ints). Used in PendingAction and ObjectFactory.

NOTE: Be very accurate while working with this class. You shouldn't store in ParamArray any classes instantiations. Reason of such restriction is pretty simple: ParamArray uses C functions malloc() and free() (instead of new and delete).
