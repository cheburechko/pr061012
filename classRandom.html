Detailed Description

Interface for getting pseudorandom numbers.

NOTE: As you can see this interface doesn't use function overloading. This is done in order to prevent bugs such like when you want get double random number from 10 to 20 range and accidently use range(10, 20) instead of needed range(10.0, 20.0).
